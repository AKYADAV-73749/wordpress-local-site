# wordpress-local-site
A simple WordPress site created locally using Gutenberg editor. Includes Home and About pages, internal navigation, and static homepage setup.
# WordPress Local Site (Practice Project)

This project demonstrates my hands-on experience with WordPress.

## Features
- WordPress installed and configured locally
- Pages created using Gutenberg block editor
- Home and About pages
- Button-based internal navigation
- Static homepage configuration
- Basic content and layout customization

## Tech Stack
- WordPress
- PHP
- HTML
- CSS
- MySQL (local)

## Notes
This site was developed locally for learning and practice purposes.
